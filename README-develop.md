# swagger-viewer for developer

## Install

```sh
yarn
```

## npm scripts

### Development

```sh
yarn dev
```

And load `dist/` as Chrome extension.

### Build

```sh
yarn build
```

And go to Chrome Web Store to register the zip.

### UnitTest

```sh
yarn test
```

### Lint

```sh
yarn lint

# dry-run
yarn lint-dry
```
