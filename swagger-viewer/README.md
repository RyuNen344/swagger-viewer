# swagger-viewer for developer

## Install

```sh
npm i
```

## npm scripts

### Development

```sh
npm run dev
```

And load `dist/chrome` as Chrome extension.

### Build

```sh
npm run build
```

And register `packages/*.zip` as Chrome extension.

### UnitTest

```sh
npm t
```

### Lint (ESLint autofix + type check)

```
npm run lint
```
